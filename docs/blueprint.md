# **App Name**: WorkoutAI

## Core Features:

- Workout Tracking: Log completed workouts and track progress over time.
- Workout Creation: Create custom workouts with exercise selection, sets, reps, and rest intervals.
- Exercise Library: Browse a database of exercises with descriptions and visual aids.
- AI Workout Generation: Use AI to generate personalized workout plans based on fitness goals and available equipment. The AI tool is a planner and selector, using data to pick the best workout.
- Progress Visualization: Visualize progress with charts and graphs tracking performance metrics.
- Social Sharing: Share workout achievements and progress with friends.

## Style Guidelines:

- Primary color: Vibrant orange (#FF6A3D), referencing the 'Novo Treino' button for focus and energy.
- Background color: Soft, desaturated orange (#F8F1EE). Dark scheme adapted with high brightness for clarity.
- Accent color: Muted red (#FF5733). It is analogous, adding slight emphasis without clashing with the theme.
- Body and headline font: 'Inter' sans-serif. Provides a modern, clean look suitable for displaying workout information clearly.
- Minimal gray icons for actions like delete, edit, and duplicate.
- Apple-inspired UI with clean spacing and subtle shadows for a modern aesthetic.
- Subtle transitions and animations when navigating between workout sections.
