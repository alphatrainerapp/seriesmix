# WorkoutAI

Seu assistente pessoal de treinos com Inteligência Artificial.

## Funcionalidades
- **Gestão de Treinos:** Organize suas divisões (A, B, C, D).
- **Combinações Avançadas:** Suporte para Biset, Triset, Supersérie e HIIT.
- **Configuração de HIIT:** Definição automática de voltas, tempo de execução e descanso.
- **Edição em Tempo Real:** Ajuste variáveis como repetições e carga diretamente na tabela.
- **IA Generativa:** Geração de resumos e variações de exercícios usando Genkit e Google Gemini.

## Tecnologias
- Next.js 15
- React 19
- Tailwind CSS
- Shadcn UI
- Genkit (IA)
- Lucide React Icons

## Como rodar localmente
1. Instale as dependências: `npm install`
2. Configure seu `.env` com a `GEMINI_API_KEY`.
3. Rode o servidor de desenvolvimento: `npm run dev`
