# WorkoutAI 🚀

Seu assistente pessoal de treinos com Inteligência Artificial, desenvolvido com **Next.js 15**, **Genkit** e **Shadcn UI**.

## 🔗 Repositório
Suba seu projeto para o GitHub e cole o link aqui para compartilhar com o mundo!

## 🚀 Como subir para o seu GitHub (Guia Rápido)
1. Crie um repositório novo e vazio no [GitHub](https://github.com/new).
2. Abra o terminal na pasta do projeto e execute:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: WorkoutAI v1.0"
   git branch -M main
   git remote add origin https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git
   git push -u origin main
   ```

## 🛠 Funcionalidades de Elite
- **🤖 IA Generativa:** Geração de planos de treino ultra-personalizados e variações de exercícios.
- **🔍 Busca Inteligente:** Filtros Apple-style por divisão (A, B, C) e grupos musculares com ícones.
- **📂 Gestão de Biblioteca:** Salve sessões personalizadas e modelos favoritos em pastas.
- **📊 Métricas Visuais:** Gráficos de volume de treino por grupo muscular usando Recharts.
- **🌓 Dark Mode:** Interface premium que se adapta totalmente (incluindo popups).
- **📱 Mobile First:** UI otimizada para uso intenso na academia via smartphone.

## 💻 Stack Tecnológica
- **Framework:** Next.js 15 (App Router)
- **IA:** Google Genkit + Gemini 2.5 Flash
- **Estilo:** Tailwind CSS + Shadcn UI (Custom Premium Theme)
- **Ícones:** Lucide React
- **Gráficos:** Recharts

## ⌨️ Comandos Git úteis
- `git status`: Verifica o que foi alterado.
- `git add .`: Prepara as mudanças para o commit.
- `git commit -m "mensagem"`: Salva as alterações localmente.
- `git push`: Envia as mudanças para o GitHub.
- `git pull`: Baixa as mudanças do GitHub.
